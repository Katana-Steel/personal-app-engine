About Katana Steel

I started learning BASIC at 13 on my own and from library book. From there I've been writing software since I was 16. Starting with simple D&D tools in Word Basic to generate game character sheets.
And as many a teenager I wanted to create VideoGames and got into C and C++.
With my first IT job I got into PHP and QT to create a simple CRM System from scratch building from requirement to requirement, a sort of BDD or FDD(feature drivent development).

Recently I've started tinkering with the defunct/abandoned game enging LithTech with the hope that at somepoint I can play the game F.E.A.R. naitively on my OS of choice.
By finishing up the migration of xfburn to GTK+3 framework I've also taken up maintenance of this project, which is distibuted with pretty much all the Linux distributions that ships the XFCE desktop.

This web site is the home for some of my pet projects and in the menu bar you will find links to the public git repositories.
If you have a promo link it will store a cookie with the link id and add it to the menu bar for the next 30 days or until you clear your cookies.
